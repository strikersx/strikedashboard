# Aviso de Privacidade do StrikeLab

> **Versão:** 1.0 · **Data:** Maio 2026 · **Responsável:** Striker's House Lda

---

## O que é o StrikeLab?

O StrikeLab é um programa de gamificação da Striker's House que transforma os teus treinos em pontos, níveis e prémios. Quanto mais treinas, mais ganhas.

## Que dados pessoais utilizamos?

| Dado | Para quê | De onde vem |
|------|---------|-------------|
| ID de cliente | Identificar o teu perfil | Yogo Booking |
| Presenças em aulas | Contabilizar pontos | Check-in Yogo |
| Telefone | Ligação WhatsApp ↔ perfil | Yogo / WhatsApp |
| Email | Identificação alternativa | Yogo |
| Instagram (opcional) | Verificação de identidade | Tu, via código de verificação |
| Ano de nascimento | Protecção de menores | Yogo |

## Bases legais

Utilizamos os teus dados com base no **consentimento** (Art. 6(1)(a) RGPD). Podes retirar o consentimento a qualquer momento.

## Consentimentos

Ao aderires ao StrikeLab, autorizas:

1. **Treino** — Utilizar as tuas presenças para calcular pontos
2. **UGC** (opcional) — Usar conteúdo gerado por ti em comunicações
3. **Nome real** (opcional) — Mostrar o teu nome no leaderboard
4. **Broadcasts** (opcional) — Enviar informações sobre o programa

Cada um pode ser activado ou desactivado independentemente.

## Menores (menores de 18 anos)

- **Idade mínima:** 13 anos
- **13–17 anos:** Necessária autorização dos pais ou encarregado de educação
- Sem data de nascimento registada no sistema, não é possível aderir

## Durante quanto tempo guardamos os dados?

- **Presenças:** 24 meses
- **Pontos/XP:** Mensal (reset) / XP vitalício
- **Perfil:** Até pedires o apagamento
- **Snapshots mensais:** 12 meses

## Os teus direitos

Tens direito a:

- **Acesso** — Saber que dados temos sobre ti
- **Rectificação** — Corrigir dados incorrectos
- **Apagamento** — Pedir a eliminação dos teus dados ("direito ao esquecimento")
- **Portabilidade** — Receber os teus dados em formato digital
- **Oposição** — Opor-te ao tratamento dos dados

Para exercer qualquer direito, fala com o Marcelo na recepção ou envia email para **ricardo@strikershouse.pt**.

## Partilha de dados

Não partilhamos os teus dados com terceiros para fins de marketing. Os dados são processados exclusivamente para operar o StrikeLab.

**Processadores técnicos:**
- Vercel (hosting — EUA, com cláusulas contratuais padrão)
- Turso (base de dados — UE)

## Segurança

- Ligações cifradas (HTTPS/TLS)
- Base de dados cifrada em repouso
- Acesso restrito a administradores autorizados
- Audit log de todas as operações
